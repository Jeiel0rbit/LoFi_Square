# Instalação

```bash
yarn add electron-builder --dev
```

#

```bash
yarn install
```

#

- Windows e Linux

```bash
yarn build:win-linux
```

#

- Mac

```bash
yarn build:mac
```

# Atualização

Desinstale o programa anterior por completo.
No Linux por exemplo seria o comando:

```bash
sudo dpkg --purge lofi-radio
```

> Se baixado `.deb`.
